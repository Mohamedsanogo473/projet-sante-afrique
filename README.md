# Projet 2 : Impact des dépenses de sante publique en Afrique

## Objectif
Analyser l effet des depenses publiques de sante sur la mortalite
infantile et l esperance de vie dans 54 pays africains (2000-2022).

## Outils utilises
- SQL (DB Browser for SQLite)
- Stata 15
- Power BI Desktop

## Variables
- Depenses sante (% PIB)
- Mortalite infantile (pour 1000 naissances)
- Esperance de vie (annees)
- Mortalite maternelle (pour 100000 naissances)

## Sources de donnees
- Banque Mondiale
